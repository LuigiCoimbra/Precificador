# Precificador
```
Serviço desenvolvido para realizar a precificação dos produtos vendidos em marketplaces,
para obter o melhor preço possivel e ficar pareo com a competição
```

## Requirements
```
Node.js
Versão LTS
https://nodejs.org/en
```

## Project Setup
```
npm install
```

### Run Service
```
npm start
```
